# Luxury Watch Landing Page – LuxChronos

## Overview
A modern and elegant landing page designed for a luxury watch brand. The project focuses on delivering a premium user experience with a clean layout and high-end visual design.

## Features
- Responsive design (works on all devices)
- Elegant typography and dark luxury theme
- Smooth navigation and UI interactions
- Sections: Collections, Craftsmanship, Testimonials

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project demonstrates front-end development skills and UI/UX design for luxury brand websites.
